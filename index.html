Калькулятор процентной ставки

Функциональность:


Входные данные:

  • сумма, руб (1-100: +1%, 100-200: +2%, 200+: +3%)
  • срок, лет (1: +1% / 2: +2%)
  • ВИП клиент? (true: +1% / false: +0%)
  • есть ли уже открытый вклад? (true: -3% / false: 8%)

Выходные данные: процентная ставка = сумме % по каждому параметру
